# QSAR ML Analysis

Проект посвящён исследованию методов машинного обучения для решения QSAR-задач прогнозирования биологической активности и селективности соединений на основе молекулярных дескрипторов.

## Задачи
- Регрессия IC50
- Регрессия CC50
- Регрессия SI
- Классификация IC50
- Классификация CC50
- Классификация SI

## Используемые методы
- Linear Regression
- Lasso Regression
- Decision Tree
- Random Forest
- CatBoost
- SVM / SVR

## Стек
Python, pandas, scikit-learn, CatBoost, matplotlib, seaborn
